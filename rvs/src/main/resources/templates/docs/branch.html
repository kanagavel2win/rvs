Getting Started

Welcome to RVSLS Website documentation.

This guide provides instructions on how to use the software effectively. It is designed for administrators, support staff, and end-users.

You are seeing the Administration link on the left side of the website. Dropdown it. You can see the three links,

  • Branches

  • Asset Management

  • HR Management

    • First, you have to click Branches. Now, you can see the site like below,



    Click on your plus-icon. The plus-icon shows Details of the Specific person like, the person's Branch Name, Position and Status(Active/Inactive).



    Next,The Check Box is used for select multiple of Branches to Export.



    Next, Under your Branch list, If you are click branch name(For example, Coimbatore) you can view youor full proflie.
    Here, you can edit and update your profile, Address.

    Next you can see your contacts and Documents. Both contacts and Documents have an Edit/Delete and Update options and also have add option. If you want to add/edit means, you have to change yourself.

    Next, Export Menu have to dropdown the below list.
    1. Print
    2. Csv (Comma Separated Values file)
    3. Pdf
    4. Copy
